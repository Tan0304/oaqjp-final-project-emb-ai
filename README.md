# Emotion Detection Final Project

This project implements an emotion detection application using Flask and Watson NLP services.

## Features
- Detects emotions like anger, joy, sadness, fear, and disgust from text input.
- Provides a web interface and REST API endpoint.
- Handles error cases such as blank input.

## How to Run
1. Run the Flask server: `python3 server.py`
2. Access the web UI via the provided URL or use the `/emotionDetector` API endpoint.
3. Submit text to get emotion scores and dominant emotion.

## Technologies Used
- Python 3
- Flask
- Watson NLP Emotion Predict API
